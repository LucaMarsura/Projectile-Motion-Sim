Projectile
Motion Sim

Visualize. Interact. Comprehend.

This python-programmed simulator models real-world motion of launched projectiles. Twelve user-controlled variables affect the trajectory of the projectile.

No sign-up required. Free to use.

Accurate Physics

1000 Hz integration guarantees accurate drag force modelling while maintaining a comprehensible user interface.

Interactive Controls

Twelve interactive sliders and pre-set parameter options allow the user to precisely determine launch parameters and observe real-world examples.

Rich Output

Our program exports various results, such as a comprehensive 3-point energy analysis as well as relative uncertainty propagations.

Great for Learning

This program provides an interactive method of understanding an otherwise unintuitive physics concept.
