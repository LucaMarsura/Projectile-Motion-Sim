# Projectile-Motion-Sim

A computational physics program simulating projectile motion using iterative variable estimation and nonlinear drag force integration at a frequency of 100 Hz.

Features include:

- Variable Inferences through Iterative Solving
- Live Trial Results documenting iteration progress
- Fundamental Energy Analysis
- Dynamic Relative Uncertainty Propagations

Eight total dynamic variables:

- Initial Velocity
- Initial Height
- Launch Angle
- Ambient Gravity
- Total Distance
- Total Time
- Peak Height
- Final Velocity

Five total static variables: 
- Drag Coefficient
- Fluid Density
- Projectile Mass
- Cross-Sectional Area
- Drag coefficient
- Impact Angle

Variable Inferences for:

4/4 Independent Variables Given: Initial Velocity, Initial Height, Launch Angle, Ambient Gravity

or

3/4 Independent Variables Given: Initial Velocity, Initial Height, Launch Angle, Ambient Gravity

and

1/4 Dependent Variable Given: Ambient Gravity, Total Distance, Total Time, Peak Height, Final Velocity
